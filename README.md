# Student_Performance_Analysis
A Python-based data analysis project that explores student academic performance using exploratory data analysis (EDA), factor analysis, data visualization, and at-risk student segmentation to generate actionable educational insights.

## 📌 Project Overview

This project analyzes student academic performance using Python and Exploratory Data Analysis (EDA). The objective is to identify the factors influencing student performance and provide data-driven recommendations to improve educational outcomes.

The analysis includes data cleaning, factor analysis, data visualization, at-risk student identification, and business insights.

---

## 🎯 Problem Statement

The school principal wants to understand the factors affecting student performance and identify students who require additional academic support. This project answers the following questions:

- Does parental education affect student performance?
- Does completing the test preparation course improve scores?
- What is the relationship between Math, Reading, and Writing scores?
- Which gender performs better in each subject?
- How are total scores distributed?
- Which students are at risk academically?

---

## 📂 Dataset

**Dataset:** Students Performance in Exams

The dataset contains information about students, including:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

> **Note:** The dataset is publicly available on Kaggle and is not included in this repository.

## 🔍 Key Findings

- Students whose parents have higher education levels generally achieve better academic results.
- Students who completed the test preparation course scored higher in all subjects.
- Reading and Writing scores have the strongest positive correlation.
- Male students performed better in Mathematics.
- Female students performed better in Reading and Writing.
- Most students achieved average to good academic performance.
- At-risk students were identified for early academic intervention.

---

## 💡 Business Recommendations

- Encourage participation in test preparation courses.
- Provide remedial classes for at-risk students.
- Strengthen parental involvement in education.
- Implement subject-specific learning strategies.
- Monitor student performance regularly using data analytics.
